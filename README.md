## Development

### Prerequisites
- Node.js 18+ and npm

### Setup and Installation

1. Clone the repository
```bash
git clone https://github.com/Harsh01Y/stemtec.git
cd stemtec
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```

### Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
